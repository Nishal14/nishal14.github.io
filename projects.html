Step-Free Arithmetic Transformers

Mechanistic Interpretability Study

GitHub • Writeup

Built a reproducible research framework for training compact arithmetic transformers (0.66M to 10M parameters) on hierarchical mathematical reasoning tasks using PyTorch and mixed-precision training. Applied mechanistic interpretability techniques, including attention-head ablation, linear probing, and out-of-distribution evaluation, to identify parenthesis-sensitive computation circuits underlying arithmetic reasoning.

Mapping Hallucinations in Tiny Aya

Multilingual Hallucination Detection Study

Participated in Expedition Tiny Aya, a global research initiative by Cohere Labs, collaborating within a five-member international team to investigate multilingual hallucinations in Tiny Aya models. Applied black-box metrics (Hallucination Rate, Self-Consistency, Variance, Cross-Model Disagreement, Prompt Sensitivity Score, etc.) alongside white-box methods to detect hallucination signals using Python-based pipelines.

Aletheia

Evidence-Grounded LLM Assistant

GitHub

Engineered a transparent retrieval-augmented reasoning pipeline using techniques from Loki (claim decomposition), LM-Polygraph (verbalized confidence estimation), FIRE (adaptive retrieval), and OpenFactCheck (evidence-grounded verification), integrating them into a unified FastAPI, Qdrant, Ollama, and Next.js system. Developed a self-enriching RAG architecture that automatically retrieves and indexes arXiv and Wikipedia content when retrieval coverage is insufficient, enabling continual expansion of the local knowledge base.

Grokking Feasibility Boundaries in Transformers

Generalization Dynamics Study

GitHub • Writeup

Constructed controlled interpolation–extrapolation benchmarks for modular arithmetic to isolate the effect of train–test overlap on grokking behaviour in compact transformer models. Investigated delayed generalization through representation geometry, representation rank collapse, and attention entropy, identifying empirical boundaries for grokking under varying supervision regimes.

Continuum K2-Think Hackathon

AI Safety Infrastructure

GitHub

Designed an AI safety system for detecting epistemic drift and contradiction accumulation across long-running LLM conversations through cumulative stance tracking and dependency analysis. Implemented a hybrid verification pipeline combining symbolic heuristics with external reasoning models to identify, validate, and visualize conversational inconsistencies.
